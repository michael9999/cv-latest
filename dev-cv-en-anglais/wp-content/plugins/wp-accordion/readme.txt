== Description ==

wordpress plugin that collects active widget and form an accordion animation

== Note ==
IF Bug exist just leave a msg or drop me an email , jcangeles16@yahoo.com =)

== Installation ==

1. Upload 'wp-accordion-izer' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. you'll see a second side bar named accordion, drop widget that you want to be accordionized
4. drop the wp-accordion widget to the main sidebar for the accordion output
